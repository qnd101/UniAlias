UniAlias Help

Basic Usage

  • Type your desired alias in the textbox.
  • Use Tab, ↑, and ↓ to navigate through completions.
  • Press Enter to select and insert the character.
  • Press Esc to close without selecting.

Quick Access

  • Press Shift+Alt+U to open the window anytime.
  • Alternatively, click the system tray icon to show the window.
  • To exit completely, right-click the system tray icon and select "Exit".

Additional Features

  • Click the "Reload" button to refresh the alias dataset
  • The program runs in the background for quick access
