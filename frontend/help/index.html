UniAlias Help

Basic Usage

  • Type your desired alias in the textbox.
  • Use Tab, ↑, and ↓ to navigate through completions.
  • Press Enter to select and insert the character.
  • Press Esc to close without selecting.

Quick Access

  • Press Shift+Alt+U to open the window anytime. (The hotkey may be changed in the Settings window)
  • Alternatively, click the system tray icon to show the window.
  • To exit completely, right-click the system tray icon and select "Exit".

Additional Features

  • Click the "Reload" button to refresh the alias dataset
  • The program runs in the background for quick access

Datasets

  • Alias data is stored in CSV files in your app data folder under:
    com.qnd101.unialias.app/dataset/
  • The app data folder is
    • Windows: %AppData%
    • Mac: $HOME/Library/Application Support
    • Linux: $XDG_DATA_HOME or $HOME/.local/share
  • CSV files should follow this format: 
    alias,character    # Example: alpha,α
  • Rules for CSV files:
    • Aliases must use ASCII characters only (no spaces)
    • Characters must be single Unicode characters
    • Be carfeful that character encoding should be UTF-8
    • Lines starting with # are treated as comments
  • Use the "Reload" button after modifying dataset files
  • Each dataset file comes with a markdown file containing information about the dataset. You may view its contents by opening the Dataset Management window.

Debug & Issues

  • If you see an issue, check the log files. Follow these instructions to find the log file.
    1. Navigate to the appropriate folder for your OS.
      • Windows: AppData/Local
      • Mac: $HOME/Library/Logs
      • Linux: $XDG_DATA_HOME or $HOME/.local/share
    2. Find com.qnd101.unialias.app/logs/
  • For more help, please visit the GitHub Issues page.
